
var x = 10;


function a(){

var x = 20;

return function(){
 this.x;
}
}


a();


这样return的是10。(this是在函数被调用时确定的，当a被调用时，a不被对象拥有，所以这里this是undefiend，因为是非严格模式，这里this是window，即window.x，全局变量x在window中是一个对象 可以在dir(window)中看到x)


如果是用 return ()=> this.x | return的是20。 


原因：箭头函数中this 一直！！指向他的爸爸function！！ 记住这句话就好了 爸爸function 爸爸函数
